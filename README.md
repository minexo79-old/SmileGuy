## 建置環境：
`python 3.7.4`
`discord.py 1.2.5`
`mysqlclient`
`一台可以跑的電腦(X`

## 建置步驟：
1. 將`setting_bot-example.json`改名成`setting_bot.json`並填入以下資料：
```js
{
	"Token" : "在Discord Developer Portal獲取的TOKEN",
	"SQLserver": "你的MySQL伺服器",
	"User":"MySQL使用者",
	"Passwd":"MySQL密碼",
	"Database":"指定資料庫",
	"Owner":"權限擁有者"
}
```
2. 打開`bot.py`之後機器人就會自動運行。
3. `setting_react.json`內有道「早」的回應，可以自行更改。  

## 管理指令：
* 模組裝載(僅權限擁有者使用):  
  * 載入: `s!load` <模組>
  * 卸載: `s!unload` <模組>
  * 重載: `s!reload` <模組>
* 刪除聊天紀錄:
  * `s!msgclear` <欲刪除訊息個數>
